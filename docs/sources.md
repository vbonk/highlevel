# Sources for HighLevel Knowledge & Best Practices  

## Official HighLevel Support & Documentation  
- **Solutions Portal** – categories like Commerce, Customer Support, Developer Resources, and more provide official guides and troubleshooting topics ([Solutions - HighLevel Support Portal](https://help.gohighlevel.com/support/solutions#:~:text=Solutions%20,Notifications%20%284)).  
- **Help Center & Knowledge Base** – articles covering API documentation, whitelabel domain setup, email sending domains, SaaS mode, and other core features ([HighLevel Support Portal](https://help.gohighlevel.com/support/home)).  
- **Email Sending Guide: Email Best Practices & Warm Up** (June 27 2025) – HighLevel’s guide on warming up domains and improving email deliverability ([Email Sending Guide: Email Best Practices & Email Warm Up](https://help.gohighlevel.com/support/solutions/articles/155000001021-email-sending-guide-email-best-practices-email-warm-up#:~:text=Email%20Sending%20Guide%3A%20Email%20Best,Practices%20%26%20Email%20Warm%20Up)).  
- **A2P 10DLC Campaign Approval Best Practices** – official tips for campaign approval and texting compliance.  
- **HighLevel API Documentation – Developer Portal** – endpoints, support details and rate limits for developers ([HighLevel API Documentation - Developer Portal](https://marketplace.gohighlevel.com/docs/#:~:text=HighLevel%20API%20Documentation%20,Skip%20to%20main%20content)).  
- **Documents & Contracts API** – instructions for automating document workflows such as e-signatures.  
- **Knowledge Base: Document & Rich-Text Support** – details on uploading DOC/DOCX/PDF files to the knowledge base and using AI to cite passages.  
- **SaaS Configurator Public APIs** – article describing available endpoints for SaaS configuration ([HighLevel API](https://help.gohighlevel.com/support/solutions/articles/48001060529-highlevel-api#:~:text=Public%20API%20Endpoints%20for%20SaaS,API%20Endpoints%20for%20SaaS%20Configurator)).  

## Communities & Forums  
- **r/gohighlevel** – Reddit community for GoHighLevel users to share experiences, ask questions, and get support ([r/gohighlevel](https://www.reddit.com/r/gohighlevel/#:~:text=r%2Fgohighlevel%20,working%20for%20any%20of%20you)).  
- **r/HighLevel** – another Reddit community dedicated to HighLevel where users post guides, tutorials, and updates.  
- **HighLevel Developers Community** – official developer community site with API docs and discussion forums.  
- **Other communities** – Facebook groups and paid communities (e.g., GoHighLevel Elite) where users share tips and support.  

## Third-Party Articles & Guides  
- **10 Best Practices for Using GoHighLevel** (Viasocket, 2024) – article listing best practices for account setup, CRM optimization, automating follow-ups, using pipelines and funnels, scheduling, marketing campaigns, and analytics ([r/gohighlevel](https://www.reddit.com/r/gohighlevel/#:~:text=10%20Best%20Practices%20for%20Using,Improve%20Campaign%20Performance%20with%20Analytics)).  
- **Email Deliverability Guides** – third-party blogs (e.g., Growthable, Infraforge) offering tips on improving email deliverability, including SPF/DKIM/DMARC setup, list hygiene, and sending frequency ([GoHighLevel API V2 Documentation](https://github.com/GoHighLevel/highlevel-api-docs#:~:text=Best%20Practices%20for%20SMS%20deliverability,deliverability%20and%20Avoiding%20SMS%20Restrictions)).  
- **API Integration Guides** – tutorials on integrating leads into HighLevel via the API using tools like Postman and best practices for using the HighLevel API.  
- **Client Onboarding & Best Practices** – LinkedIn articles or blog posts detailing best practices for onboarding clients on HighLevel and customizing the platform effectively.
