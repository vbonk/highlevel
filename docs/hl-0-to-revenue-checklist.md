# HighLevel 0-to-Revenue Checklist

This checklist provides a step-by-step sequence to build a revenue-ready HighLevel system using the Minimal Viable AI + HighLevel Implementation Plan.  
## Phase 1: Set Up AI Expert Nucleus  
- Define the AI Nucleus system prompt and memory file.  
- Include core HighLevel concepts, best practices, and Velte conventions.  
## Phase 2: Design HighLevel Architecture  
- Identify your primary business use cases and customer journey.  
- Define pipelines and stages following your naming conventions.  
- Determine tag taxonomy and create necessary tags.  
- Plan workflows: triggers, actions, modular organization.  
- Decide on custom fields and form structures.  
## Phase 3: Configure HighLevel  
- Create pipelines and stages in HighLevel.  
- Set up custom fields, forms, and surveys.  
- Build calendars with appropriate availability.  
- Add essential workflows using the AI nucleus guidance.  
- Configure communication channels (Email, SMS, phone).  
- Build funnels and landing pages for your offers.  
## Phase 4: Enable Lead Intake and Routing  
- Integrate lead sources (website forms, ads, third-party sources).  
- Assign leads to pipelines and stages automatically via workflows.  
- Apply tags and custom fields upon capture.  
## Phase 5: Implement Follow-Up Systems  
- Create immediate follow-up workflows (welcome email/SMS).  
- Build nurture sequences for short-term and long-term engagement.  
- Set up appointment reminders and confirmations.  
## Phase 6: Configure Deliverability and Integrations  
- Verify domain and setup DNS records for email (SPF, DKIM).  
- Connect Mailgun/Twilio/SMTP for email and SMS.  
- Integrate payment systems (Stripe) if needed.  
## Phase 7: Launch and Test  
- Test each form and funnel to ensure data flows correctly.  
- Verify workflows trigger as expected.  
- Ensure notifications and tasks are received.  
- Conduct a soft launch with a small audience.  
## Phase 8: Go Live and Monitor  
- Launch your marketing campaigns and drive traffic.  
- Monitor pipeline movement, conversions, and communication deliverability.  
- Use dashboards to track performance and identify issues.  
## Phase 9: Iterate and Optimize  
- Review results and gather feedback.  
- Adjust workflows, messaging, and funnels based on data.  
- Update the AI nucleus memory file with new insights.
